# Agro 5 Lang Hub 🌐

Бесплатный онлайн-переводчик текстов на 5 языков.  
Работает на Flask (Python), **без JavaScript** — вся логика на сервере.

---

## 🌍 Поддерживаемые языки

| Язык | Код | Интерфейс | Перевод |
|------|-----|-----------|---------|
| 🇷🇺 Русский | `ru` | ✅ | ✅ |
| 🇺🇿 Ўзбекча | `uz` | ✅ | ✅ |
| 🇬🇧 English | `en` | ✅ | ✅ |
| 🇯🇵 日本語 | `ja` | ✅ | ✅ |
| 🇰🇷 Қарақалпақ | `kaa` | ✅ | ⚠️ ограничено |

---

## ⚙️ Технологии

- **Backend:** Flask (Python)
- **Frontend:** Jinja2 + HTML + CSS (без JavaScript)
- **API перевода:** MyMemory (основной) + LibreTranslate (резервный)
- **Дизайн:** Glassmorphism, адаптивный, анимации на CSS

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/your-username/agro-5-lang-hub.git
cd agro-5-lang-hub
```

### 2. Создать виртуальное окружение (рекомендуется)

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Запустить сервер

```bash
python app.py
```

Откройте в браузере: **http://127.0.0.1:5000**

## 🔄 Как работает

1. Пользователь выбирает язык интерфейса (RU / UZ / EN / JP / QQ)
2. Вводит текст и выбирает языки перевода
3. Нажимает **«Перевести»**
4. Flask отправляет запрос в MyMemory API
5. Если MyMemory недоступен → LibreTranslate (fallback)
6. Результат отображается на странице
